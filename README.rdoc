= Jsonpipe

This is a ruby gem offer command line to parse and pretty print json from io pipe.

== Install 
  
  sudo gem install jsonpipe
  
== Use it in your shell
  
  echo '{"abc/def": [1,2,3]}' | jsonpipe
  
  curl http://api.twitter.com/1/statuses/public_timeline.json | jsonpipe
